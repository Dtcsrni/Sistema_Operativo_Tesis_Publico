# Mapa del Filesystem

## Layout objetivo
- `/opt/tesis-os`: software y wrappers del sistema.
- `/srv/tesis/repo`: clon operativo del repo canónico.
- `/srv/tesis/workspace`: trabajo activo por dominio.
- `/srv/tesis/outputs`: salidas y exportaciones.
- `/var/log/tesis-os`: logs del sistema.
- `/var/lib/tesis-os`: estado local.
- `/mnt/emmc/*`: archive, backups, corpus, datasets, exports, models, snapshots.

## Regla
No mezclar datos personales sensibles con bitacora tecnica ni con material publicable.
